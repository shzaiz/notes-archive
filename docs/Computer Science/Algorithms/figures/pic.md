# 主要内容

## 魔改前

- 第一章数据结构的基本概念（3学时） 
    - 基本概念、术语，算法和算法分析
    - 程序编写规范
- 第二章线性表（6学时）
    - 线性表的概念、操作，数组和链表实现
    - 线性表的应用（一元多项式的表式和运算）
- 第三章 栈和队列（6学时）
    - 栈和队列的定义、基本运算，顺序和链表实现，应用
- 第四章 串（3学时）
    - 串的定义、基本运算，顺序和链表实现，模式匹配算法
- 第五章 数组和广义表（3学时）
    - 数组定义及基本运算，顺序、链式存储结构，十字链表表示稀疏矩阵
- 第六章 树和二叉树（9学时）
    - 树的定义、基本术语，二叉树的定义、性质和存储，二叉树的遍历和线索二叉树，树和森林，哈夫曼树及其应用，树的计数
- 第七章 图（9学时）
    - 图的定义、术语，存储结构，遍历，连通性（最小生成树），有向无环图的应用，最短路径问题。
- 第九章 查找（6学时）查
    - 找及其性能评价的概念，顺序表、有序表、静态树表、索引顺序表的查找
    - 动态表的查找（二又排序树、AVL树、B-树、键树、哈希表）
- 第十章排序（6学时）
    - 排序的概念、评价，插入排序（简单插入、二分插入、希尔）
    - 交换排序（起泡排序、快速排序）
    - 选择排序（简单选择、堆排序）
    - 归并排序，基数排序，各种排序算法的比较与选择
- 第十一章 外部排序（3学时）
    - 外部排序的概念，多路平衡归并，置换-选择排序，最佳归并树
