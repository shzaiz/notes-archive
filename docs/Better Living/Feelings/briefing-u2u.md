# 大二上学期简报

目前的 GPA 是 4.27/5. 有三个课程没有出来: 算法导论、数据库、算法导论课程设计. 我们发现, 最坏的情况如下

- 算法导论: 4.0 * 5pts
- 数据库: 3.5 * 5pts
- 算法导论课程设计 3.5 * 2pts

这样会让我们的 GPA 变到 4.20. 这样的情况下, 本学期即使在全部课程是 4.5 的情况下, GPA 也会到达 4.26. 

下面来看最好的情况. 如果老师给的好, 

- 算法导论: 4.5 * 5pts
- 数据库: 4.0 * 5pts
- 算法导论课程设计 4.5 * 2pts

那样就会达到 4.27 的 GPA. 这样的情况下, 本学期即使在全部课程是 4.5 的情况下, GPA 也会到达 4.31.

如果老师~~良心发现~~真的把数据库给的比较高, 比如 4.5, 那么就会发现 GPA 这学期就可以是 4.30. 不过也别高兴的太早, 即使 本学期即使在全部课程是 4.5 的情况下, GPA 也只会到达 4.33.
